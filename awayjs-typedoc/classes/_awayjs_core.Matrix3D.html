Properties

_rawData

_rawData: Float32Array

A Vector of 16 Numbers, where every four elements is a column of a 4x4 matrix.

<p>An exception is thrown if the _rawData property is set to a matrix that is not invertible. The Matrix3D object must be invertible. If a non-invertible matrix is needed, create a subexport class of the Matrix3D object.</p>

  • Defined in @awayjs/core/lib/geom/Matrix3D.ts:184
