dispose

  • dispose(): void

  • Frees memory that is used to store the SpecularImage2D object.

    <p>When the <code>dispose()</code> method is called on an image, the width and height of the image are set to 0. All subsequent calls to methods or properties of this SpecularImage2D instance fail, and an exception is thrown. </p>

    <p><code>SpecularImage2D.dispose()</code> releases the memory occupied by the actual bitmap data, immediately(a bitmap can consume up to 64 MB of memory). After using <code>SpecularImage2D.dispose()</code>, the SpecularImage2D object is no longer usable and an exception may be thrown if you call functions on the SpecularImage2D object. However, <code>SpecularImage2D.dispose()</code> does not garbage collect the SpecularImage2D object(approximately 128 bytes); the memory occupied by the actual SpecularImage2D object is released at the time the SpecularImage2D object is collected by the garbage collector.</p>

    Returns void

    Overrides Image2D.dispose
