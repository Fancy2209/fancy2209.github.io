Packagecom.adobe.flascc
Classpublic class Console
InheritanceConsole flash.display.Sprite
Implements ISpecialFile

A basic implementation of a console for CrossBridge apps. The PlayerKernel class delegates to this for things like read/write, so that console output can be displayed in a TextField on the Stage.
