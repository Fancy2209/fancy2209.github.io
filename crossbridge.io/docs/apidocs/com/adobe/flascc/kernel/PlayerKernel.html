Packagecom.adobe.flascc.kernel
Classpublic class PlayerKernel
InheritancePlayerKernel Object
Implements IKernel

This is the default implementation of the Posix interface. Extending and overriding methods in this class allow you to customize the handling of the basic syscalls on top of which libc operates.
